JavaScript Programming Fundamentals

10 lessons · 16 hours · Beginner → Intermediate

JavaScript ▼
1

Variables, Data Types & Operators

var vs let vs const, primitive types (string, number, boolean, null, undefined, symbol), type coercion, operators.

Beginner
const name = "Alice"; // string let age = 25; // number const isActive = true; // boolean let score = null; // null // Template literals console.log(`Hello, ${name}! Age: ${age}`);
2

Control Flow — if/else, switch, loops

Conditionals, ternary operator, for/while/do-while loops, break/continue, for...of, for...in.

Beginner
const scores = [85, 92, 78, 95]; for (const score of scores) { if (score >= 90) console.log("A grade"); else if (score >= 80) console.log("B grade"); else console.log("C grade"); }
3

Functions — Declarations, Expressions & Arrow

Function declarations vs expressions, arrow functions, default parameters, rest parameters, return values.

Beginner
// Arrow function const add = (a, b) => a + b; // Default params const greet = (name = "World") => `Hello, ${name}!`; // Rest params const sum = (...nums) => nums.reduce((a, b) => a + b, 0);
4

Arrays & Array Methods

Array creation, push/pop/shift/unshift, map, filter, reduce, find, some, every, spread operator.

Beginner
const nums = [1, 2, 3, 4, 5]; const doubled = nums.map(n => n * 2); // [2,4,6,8,10] const evens = nums.filter(n => n % 2 === 0); // [2,4] const total = nums.reduce((s, n) => s + n, 0); // 15
5

Objects & Destructuring

Object literals, property shorthand, computed keys, Object methods, destructuring assignment, spread in objects.

Intermediate
const user = { name: "Bob", age: 30, role: "dev" }; // Destructuring const { name, age, role = "user" } = user; // Spread const updated = { ...user, age: 31, city: "Mumbai" };
6

DOM Manipulation

querySelector, createElement, appendChild, innerHTML vs textContent, classList, dataset, event delegation.

Intermediate
const btn = document.querySelector('#myBtn'); const list = document.querySelector('#list'); btn.addEventListener('click', () => { const li = document.createElement('li'); li.textContent = 'New item'; list.appendChild(li); });
7

Events & Event Handling

addEventListener, event object, stopPropagation, preventDefault, event delegation pattern, custom events.

Intermediate
// Event delegation document.querySelector('ul').addEventListener('click', (e) => { if (e.target.tagName === 'LI') { e.target.classList.toggle('done'); } }); // Custom event const event = new CustomEvent('userLogin', { detail: { userId: 1 } }); document.dispatchEvent(event);
8

Scope, Hoisting & Closures

Global vs function vs block scope, hoisting of var/function, temporal dead zone, closure use cases.

Intermediate
// Closure — counter factory function makeCounter(start = 0) { let count = start; return { increment: () => ++count, decrement: () => --count, value: () => count }; } const counter = makeCounter(5); counter.increment(); // 6
9

Error Handling & Debugging

try/catch/finally, Error types, throwing custom errors, console methods, debugger statement.

Intermediate
function divide(a, b) { if (b === 0) throw new Error("Division by zero!"); return a / b; } try { console.log(divide(10, 0)); } catch (err) { console.error("Caught:", err.message); } finally { console.log("Done"); }
10

Fetch API & Working with JSON

Fetch API, GET/POST requests, JSON.parse/stringify, response handling, async data loading patterns.

Intermediate
async function getUsers() { const res = await fetch('https://api.example.com/users'); if (!res.ok) throw new Error(`HTTP ${res.status}`); const data = await res.json(); return data.users; } getUsers().then(users => console.log(users));

Advanced JS & Async Programming

9 lessons · 18 hours · Intermediate → Advanced

Advanced JS ▼
1

Execution Context & Call Stack

How JavaScript executes code, global/function execution context, call stack, stack overflow, single-threaded nature.

Intermediate
// Stack trace function third() { throw new Error("Oops!"); } function second() { return third(); } function first() { return second(); } // Call stack: first → second → third → Error
2

Prototype Chain & Inheritance

__proto__, Object.create, class syntax, extends, super, method overriding, instanceof.

Advanced
class Animal { constructor(name) { this.name = name; } speak() { return `${this.name} makes a sound.`; } } class Dog extends Animal { speak() { return `${this.name} barks.`; } } const d = new Dog("Rex"); d.speak(); // "Rex barks."
3

Event Loop, Microtasks & Macrotasks

Web APIs, callback queue, microtask queue, event loop mechanism, Promise vs setTimeout ordering.

Advanced
console.log('1'); // sync setTimeout(() => console.log('2'), 0); // macrotask Promise.resolve().then(() => console.log('3')); // microtask console.log('4'); // sync // Output: 1, 4, 3, 2
4

Promises Deep Dive

Promise states, chaining, Promise.all, Promise.allSettled, Promise.race, error propagation.

Intermediate
const p1 = fetch('/api/user'); const p2 = fetch('/api/posts'); // Parallel fetch const [user, posts] = await Promise.all([p1.then(r=>r.json()), p2.then(r=>r.json())]); // Handle mixed results const results = await Promise.allSettled([p1, p2]);
5

Async/Await Patterns

Converting promise chains to async/await, error handling, parallel vs sequential execution, async IIFE.

Intermediate
// Sequential (slow) const a = await fetchA(); const b = await fetchB(); // Parallel (fast) const [a, b] = await Promise.all([fetchA(), fetchB()]); // Error handling try { const data = await fetchData(); } catch (e) { console.error(e); }
6

Generators & Iterators

Symbol.iterator, custom iterables, generator functions, yield, async generators for streaming data.

Advanced
function* range(start, end, step = 1) { for (let i = start; i <= end; i += step) { yield i; } } for (const n of range(1, 10, 2)) { console.log(n); // 1, 3, 5, 7, 9 }
7

Modules (ES Modules & CommonJS)

import/export, named vs default, dynamic import(), tree shaking, module bundling concepts.

Intermediate
// math.js export const add = (a, b) => a + b; export default function multiply(a, b) { return a * b; } // main.js import multiply, { add } from './math.js'; // Dynamic const { add } = await import('./math.js');
8

WeakMap, WeakSet, Symbol & Proxy

Memory-safe collections, unique keys with Symbol, Proxy for metaprogramming, Reflect API.

Advanced
const cache = new WeakMap(); function memoize(fn, obj) { if (!cache.has(obj)) { cache.set(obj, fn(obj)); } return cache.get(obj); }
9

Performance & Memory Management

Memory leaks, garbage collection, debounce/throttle, Web Workers for off-thread computation.

Advanced
// Debounce function debounce(fn, delay) { let timer; return (...args) => { clearTimeout(timer); timer = setTimeout(() => fn(...args), delay); }; } const search = debounce(fetchResults, 300);
